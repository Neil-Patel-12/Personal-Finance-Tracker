# Personal-Finance-Tracker

This is a Personal Finance Tracker. Has the ability to track and log different transactions.
Organize them, get summaries of income and expenses, and view summary in a graph to see how
much money you are spending on a daily basis.

# How it works

• Used matplotlib to plot and see the line graph <br>
• Pandas allowed to easily categorize and search for data within the csv files <br>
• Data_entry.py – collect data from the user, input statements and validating different data <br>
• Main.py – handles the main flow of the program <br>
• CSV file – Where all the data is stored. It can easily be imported into something like Excel or Google Shsets, act as the databas <br>
